Vectores de Ataque: Ingeniería Social

Psicología de la Estafa

Las redes de estafas de tareas operan bajo principios de Garantía Inversa: primero entregan un beneficio real para desarmar el sistema crítico del usuario, para luego exigir desembolsos mayores mediante presión psicológica.

Ciclo de Vida del Phishing

Uso de dominios efímeros (.top, .online) controlados por Cloudflare para evadir análisis automatizados. Estos sitios mueren en cuestión de días, dificultando el rastreo forense post-ataque.

Gobernanza y Estándares

Alineación con marcos de trabajo globales para la protección de activos de información.

ISO/IEC 27001

Enfoque en el Control A.7.2.2 (Concienciación), vital para mitigar el error humano en estafas de ingeniería social.

COBIT 2019

Gestión de Riesgos de TI (EDM03). Asegurar que el apetito de riesgo y la tolerancia se mantengan en niveles gobernados.

NIST CSF

Marco de: Identificar, Proteger, Detectar, Responder y Recuperar ante incidentes de fraude digital.

Ley 30096

Marco legal peruano para delitos informáticos, estableciendo penas severas para la interceptación y el fraude.
